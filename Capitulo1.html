Capítulo 1


Elementos básicos

Estos son los elementos de esquema básico de una página HTML codificado:
El DOCTYPE declaración define el tipo de de documento como HTML.
El texto entre html y /html define un documento HTML.

Hay varias versiones de HTML incluyendo HTML, HTML 2.0, HTML 4.01, XHTML, y HTML5.
El texto entre <head> y <head> proporciona a la información sobre el documento.
El texto entre <title> y </title> proporciona al título para el documento.
El texto entre <h1> y </h1> describe un subtítulo en el documento.
El texto entre <p> y </p> describe un párrafo en el documento.

Elementos más específicos

Enlaces se pueden ser definidas con la etiqueta a.
El destino del enlace es el href attribute.
Para Definir imágenes HTML, utilice la etiqueta img.
Los atributos se proporcionan en la etiqueta img son el archivo de origen src, texto alternativo alt y el tamañ width y height.

Más información sobre los atributos:
Elementos HTML pueden haber encajado características llamados atributos. Los atributos proporcionan información adicional sobre un elemento. Ellos siempre están especificados en la etiqueta de inicio.
Tipos de atributos son el Atributo Lang (para declarar un idioma), el Atributo Título (que pueden ser declerado en el elemento p), el Atributo href (Links HTML que están definido con la etiqueta a), el Atributo Size, el Atributo alt (define el texto alternative que puede ser usando quando un elemento HTML no puede ser demonstrado).
Algos sugestiónes están que siempre use los atributos lowercase y siempre cotización valores de los atributos.

Creando formateo y separación

Insertando un salto de línea:
El <hr> etiqueta crea una línea horizontal en una página HTML y se puede utilizar para separar el contenido.
Utilice <br> si quieres una nueva línea sin necesidad de iniciar un nuevo párrafo. <br> no creará una línea visible.

Definiciónando texto preformateado
Añadir un <pre> alrededor del texto que tiene un formato fijo y el ancho. Esto preserva ambos espacios y saltos de línea. Es útil para el texto, como los poemas.
